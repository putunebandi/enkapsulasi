# Tugas PBO: Implementasi Enkapsulasi

Repositori ini berisi kumpulan program dasar Java untuk memenuhi tugas mata kuliah **Pemrograman Berorientasi Objek** materi Enkapsulasi.

## Isi File
1. `Anggota.java` : Class model untuk Anggota Koperasi dengan aturan limit dan batas minimal angsuran.
2. `TestKoperasi.java` : Program pengujian statis untuk class Anggota.
3. `ScannerTestKoperasi.java` : Program pengujian interaktif dengan scanner dan perulangan hingga hutang lunas.
4. `EncapDemo.java` : Class model dasar yang mendemonstrasikan validasi rentang umur (18-30 tahun).
5. `EncapTest.java` : Program pengujian untuk class EncapDemo.
